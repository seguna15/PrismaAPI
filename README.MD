## Prisma Commands

***

> to push new schema changes

```zsh || bash
npx prisma db push
```

> to view the schema

```zsh || bash
npx prisma studio
```

> to run migration in dev env

```zsh || bash
npx prisma migrate dev --name init
```

> to seed database

```zsh || bash
npm run seed
```
